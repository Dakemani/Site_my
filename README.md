# Задание 1.1 - Блок-схема "Приготовление чая"

## Описание
Простая блок-схема процесса приготовления чая с использованием Mermaid.

## Блок-схема

```mermaid
graph TD
    A[Начать] --> B[Вскипятить воду]
    B --> C{Есть ли чай?}
    C -->|Да| D[Заварить чай]
    C -->|Нет| E[Купить чай]
    E --> D
    D --> F[Налить чай]
    F --> G[Конец]

    # Задание 1.2 - Диаграмма последовательности "Заказ такси"

## Описание
Диаграмма последовательности, показывающая взаимодействие между участниками процесса заказа такси.

## Диаграмма последовательности

```mermaid
sequenceDiagram
    title Процесс заказа такси
    participant Клиент
    participant Приложение
    participant Сервер
    participant Водитель

    Note over Клиент: Начало процесса
    Клиент->>Приложение: Вызов такси
    activate Приложение
    Приложение->>Сервер: Запрос на поиск водителя
    activate Сервер
    
    Сервер->>Водитель: Новый заказ
    activate Водитель
    Водитель->>Сервер: Принятие заказа
    deactivate Водитель
    
    Сервер->>Приложение: Данные водителя
    deactivate Сервер
    Приложение->>Клиент: Заказ принят
    deactivate Приложение
    
    Note over Клиент, Водитель: Водитель едет к клиенту
    Водитель->>Клиент: Забирает клиента
    Note over Клиент: Поездка началась
