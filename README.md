# Задание 1.1 - Блок-схема "Приготовление чая"

## Описание
Простая блок-схема процесса приготовления чая с использованием Mermaid.

## Блок-схема

```mermaid
graph TD
    A[Начать] --> B[Вскипятить воду]
    B --> C{Есть ли чай?}
    C -->|Да| D[Заварить чай]
    C -->|Нет| E[Купить чай]
    E --> D
    D --> F[Налить чай]
    F --> G[Конец]
